Como Usar

Siga os passos abaixo para instalar e começar a usar nossas ferramentas

Baixe a Ferramenta

Escolha a ferramenta compatível com seu sistema operacional e faça o download do repositório GitHub.

Ver Ferramentas

Execute o Instalador

Após o download, execute o arquivo de instalação e siga as instruções na tela.

# Para Windows Setup-WinMaintenanceToolkit.exe # Para macOS sudo bash ./install_office_manager.sh

Conceda as Permissões Necessárias

As ferramentas precisam de permissões administrativas para funcionar corretamente.

No Windows, clique em "Sim" quando solicitado pelo UAC. No macOS, insira sua senha quando solicitado.

Comece a Usar

Após a instalação, abra a ferramenta e explore as diferentes funcionalidades disponíveis.

Consulte a documentação para obter informações detalhadas sobre cada recurso.

Perguntas Frequentes

Respostas para as dúvidas mais comuns sobre nossas ferramentas

As ferramentas são gratuitas? +
Sim, todas as nossas ferramentas são de código aberto e gratuitas para uso pessoal e comercial, sob a licença MIT.
Preciso ter conhecimentos técnicos para usar? +
Não, nossas ferramentas foram projetadas para serem intuitivas e fáceis de usar, mesmo para usuários sem conhecimentos técnicos avançados. Cada função vem com instruções claras.
As ferramentas são seguras para usar? +
Sim, todo o código é aberto e pode ser verificado por qualquer pessoa. Não coletamos dados pessoais e todas as operações são realizadas localmente no seu computador.
Posso contribuir para o desenvolvimento? +
Absolutamente! Contribuições são bem-vindas. Você pode enviar pull requests no GitHub, relatar problemas ou sugerir novos recursos. Consulte o arquivo CONTRIBUTING.md em cada repositório para obter mais informações.
Com que frequência as ferramentas são atualizadas? +
Lançamos atualizações regularmente, geralmente uma vez por mês para correções de bugs e pequenas melhorias, e atualizações maiores a cada trimestre para novos recursos. Você pode acompanhar o desenvolvimento pelo GitHub.
